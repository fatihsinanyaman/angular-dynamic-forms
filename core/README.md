# NG Dynamic Forms Core

## Installation
```
npm i @ng-dynamic-forms/core -S
```

## Import
```
@NgModule({

    imports: [
        ReactiveFormsModule,
        DynamicFormsCoreModule
    ]
})

export class AppModule {}
```
